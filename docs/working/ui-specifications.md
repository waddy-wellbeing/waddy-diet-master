# BiteRight UI/UX Specifications

## Color Palette

### Primary Colors
```css
--primary: #7C3AED;        /* Purple - main actions */
--primary-light: #A78BFA;  /* Light purple - hover states */
--primary-dark: #5B21B6;   /* Dark purple - pressed states */
```

### Status Colors
```css
--success: #10B981;        /* Green - on track, completed */
--warning: #F59E0B;        /* Amber - caution, almost there */
--danger: #EF4444;         /* Red - over budget, errors */
--info: #3B82F6;           /* Blue - informational */
```

### Neutral Colors
```css
--background: #FFFFFF;
--surface: #F9FAFB;
--border: #E5E7EB;
--text-primary: #111827;
--text-secondary: #6B7280;
--text-muted: #9CA3AF;
```

### Macro Colors (Consistent across app)
```css
--protein: #F97316;        /* Orange */
--carbs: #3B82F6;          /* Blue */
--fat: #8B5CF6;            /* Purple */
--fiber: #22C55E;          /* Green */
--calories: #EC4899;       /* Pink */
```

```css
--text-xs: 0.75rem;    /* 12px - labels, captions */

```

## Mobile Gestures

### Meal Card Swipes
- **Swipe Right (30%+)**: Quick log - "I ate this"
  - Green reveal with checkmark
  - Haptic feedback on threshold
  
- **Swipe Left (30%+)**: Show alternatives
  - Purple reveal with swap icon
  - Opens alternative recipes sheet

### Pull to Refresh
- Dashboard, Nutrition page
- Custom animation with logo

### Bottom Sheet Dismiss
- Drag down to close
- Velocity-based dismiss
